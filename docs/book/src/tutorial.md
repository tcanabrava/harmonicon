# Guided Tour

**Main Menu → Tutorial** is a quick, hands-off tour: click it and
Harmonicon automatically walks through the Main Menu, Play, Select Mode,
Options, Theme, Lessons, and Generate Jam screens, spending a few seconds
on each with a caption explaining what it's for, before returning you
right back to where you started.

![Guided tour overlay](images/tutorial-overlay.png)

- A **step counter** ("Step 3 of 7") in the overlay shows your progress
  through the tour.
- The underlying page changes beneath the overlay exactly as if you'd
  clicked through it yourself, but the overlay blocks clicks on the page
  behind it — you're not meant to interact mid-tour, just read and watch.
- Click **Skip Tutorial** at any point to end the tour immediately and
  return to the Main Menu right away, instead of waiting for the remaining
  steps.

This is meant as a fast first orientation, not a substitute for the rest of
this book — it covers *what each screen is*, not *how to use it in depth*.
For that, see the dedicated chapter for whichever screen you want to know
more about (linked from [The Main Menu](main-menu.md)).
